# 📊 CodeAlpha - Data Visualization on Netflix Dataset

## Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship. The objective is to create meaningful visualizations from the Netflix Titles dataset using Python.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations

- Movies vs TV Shows (Bar Chart)
- Movies vs TV Shows (Pie Chart)
- Top 10 Countries
- Most Common Content Ratings
- Distribution of Release Years
- Content Added by Year
- Movie Duration Distribution
- Top 10 Most Featured Actors

## Key Insights

- Movies are more common than TV Shows.
- The United States has the highest number of Netflix titles.
- Most Netflix content was released after 2010.
- Netflix expanded rapidly after 2015.
- Most movies have durations between 80 and 120 minutes.

## Conclusion

This project demonstrates how data visualization can transform raw data into meaningful insights and support better decision-making using Python.
